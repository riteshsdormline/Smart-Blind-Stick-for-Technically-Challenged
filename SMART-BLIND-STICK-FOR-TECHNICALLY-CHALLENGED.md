# Smart-Blind-Stick-for-technically-challenged

A low-cost smart cane to assist visually impaired and technically-challenged users with obstacle detection, alerting, and real-time location tracking. This repository contains the prototype design, circuit diagrams, sensor measurements, and documentation for a smart blind stick that uses Arduino, ultrasonic/IR sensors, GPS (NEO-6M), and GSM (SIM800L) modules.

Reference / background: Smart Blind Stick for Visually Impaired Navigation — IEEE article (for context)
https://ieeexplore.ieee.org/document/11386629

## Short description

This project aims to provide an affordable and easy-to-use assistive device for visually impaired or technically-challenged users. It uses ultrasonic and infrared sensors for obstacle detection, provides audio/haptic feedback, and can send location and alerts to caregivers using GSM SMS and GPS coordinates. The design focuses on low cost, simple operation, and real-time caregiver notifications.
